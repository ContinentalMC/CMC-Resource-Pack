# Models Section:

Example:
grapplinghook.json
```{
    "parent": "item/generated",
    "textures": {
	    "layer0": "continentalmc:item/grapplinghook"
    },```
Parent sets the parent of the item
Ex: item/generated 

layer0 sets the texture

continentalmc = folder continentalMC, if it is under textures it will automatically go to the textures/{block/item/other} to {item}.png
